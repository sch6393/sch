### 계산
* `%user`가 높다면 CPU, `%iowait`가 높다면 I/O의 부하가 있음
  >I/O에 대한 더 자세한 내용 ➞ [iostat](../iostat/README.md)

<br>

### 옵션
|옵션|설명|
|-|-|
|`-A`|상세 정보|
|`-b`|I/O 상태 정보|
|`-B`|페이징 통계 정보|
|`-d`|디스크 작업 확인|
|`-q`|대기열 작업 확인|
|`-r`|메모리와 스왑 공간의 상태|
|`-v`|커널 테이블과 파일에서의 [inode](../../linux/inode/README.md) 상태|
|`-w`|시스템의 Switching 상태|
|`-W`|시스템의 Swapping 상태|
